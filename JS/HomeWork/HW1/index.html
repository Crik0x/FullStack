Var, Let та Const - Відмінності

Що таке var?

var - це ключове слово для оголошення змінних у JavaScript. Змінні, оголошені за допомогою var, мають функціональну або глобальну область видимості і можуть бути повторно присвоєні.

Що таке let?

let дозволяє оголошувати змінні з блоковою областю видимості. На відміну від var, змінна, оголошена через let, не може бути повторно оголошена в тій самій області.

Що таке const?

const використовується для оголошення змінних, які не можуть бути змінені. Проте об'єкти та масиви, оголошені через const, можуть змінюватися.

Декларація vs Ініціалізація

Декларація - це створення змінної, тоді як ініціалізація - це присвоєння значення змінній.
